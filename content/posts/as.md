## 国内几大骨干网介绍

目前国内有三大运营商：**电信、联通、移动**。

| 运营商 | 骨干网名称 | 自治系统编号（AS号） |
| :--- | :--- | :--- |
| **中国电信** | 163 (CHINANET) | AS4134 |
| | CN2 (CNCN) | AS4809 |
| **中国联通** | 169 (China Unicom BackBone) | AS4837 |
| | A网 (Industrial Internet Backbone) | AS9929 |
| **中国移动** | CMNET (境内骨干网) | AS9808 |
| | CMI (境外骨干网) | AS58453 |

**科研教育用途小型骨干网：**

* **CERNET** (教育网，主用于高校): AS4538
* **CSTNET** (科技网): AS7497

> **注意：**
> 1.  如果您国内用的是手机运营商或普通家用宽带，对应 AS 号是：电信 **AS4134**，联通 **AS4837**，移动 **AS9808**。
> 2.  移动是个怪胎，普通线路不能直接连接海外，必须用 **AS58453** (CMI) 才行。

## 骨干网详情介绍

### AS4134 CHINANET 中国电信163骨干网

* **宽带业务范围：** 普通家用宽带、商用宽带、政企宽带
* **海外加速专有业务：** 163精品网套餐（上海地区）
* **已知出口：** 北京、上海、广州
* **特点：** 全国规模最大的骨干网，享有最大的国际出口。一般性电信宽带或商宽访问境外网站，如果对方 ISP 没有购买电信的 CN2 Transit，则走此骨干网。

### AS4809 CNCN(CN2) 中国电信第二代骨干网

* **宽带业务范围：** 家用游戏及海外加速宽带、商用跨国优化宽带、政企宽带
* **海外加速专有业务：** CN2国际精品网套餐（覆盖几乎全国）
* **已知出口：** 北京、上海、广州、乌鲁木齐
* **重点：CN2 分类**
    * **CN2 GIA：** 归属电信，拥有独立的**回国线路**，目前负载较轻。
    * **CN2 GT：** CN2 产品线中的 Global Transit 产品，拥有国际线路，但**没有回国路线**。
* **特点：** 技术先进，灵活性高，一般对海外聊天、游戏有较高需求的会使用。目前是国内到国际网络高峰期能提供最好速率和体验的骨干网之一。

### AS4837 China Unicom BackBone 中国联通骨干网

* **宽带业务范围：** 家宽、商宽、政企宽带
* **已知出口：** 北京、上海、广州
* **特点：** 除非访问的对方 ISP 购买了电信 CN2 Transit/联通的 CU Premiun，否则一律走 169 骨干网。出国拥堵程度小于电信 163，但总体速度和延时可靠性不如 CN2。因价格便宜，是游戏爱好者（国际服玩家）及普通外教课程需求者的首选宽带之一。

### AS9929 CHINA UNICOM Industrial Internet Backbone 中国联通工业互联骨干网

* **宽带业务范围：** 商业宽带、政企宽带
* **海外加速服务：** 本网专做海外加速服务
* **已知出口：** 北京、上海、广州
* **特点：** 前身为网通骨干网，后改为联通 A 网，用于国际互联加速服务。价格昂贵，主要为跨国企业使用，**已无家宽业务**。

### AS9808 China Mobile 中国移动骨干网

* **宽带业务范围：** 家宽、商宽、政企宽带
* **已知出口：** 北京、上海、广州
* **特点：** AS9808 为移动**境内**骨干网，所有出境流量通过 **AS58453 (CMI)** 与外网互联。

### AS58453 CMI 中国移动国际骨干网

* **特点：** 移动的**国际段骨干网**，现已不再具有国际出口优势，目前三网中只略好于电信 163。

### AS4538 CERNET 中国教育研究网络

* **宽带业务范围：** 各大高校的校园网和部分大型国内云服务提供商
* **已知出口：** 北京清华大学
* **特点：** 不服务于家宽、商宽、政企。网传效果不如 CN2。

---

**总结（骨干网）：**
CN2 GIA 是理论上可以包容所有网络环境，提供稳定且长期服务搭建节点或建站的**不二之选**，但价格是普通 VPS 的 10 倍以上。

## 国内ISP与国际ISP的互联详情

### 📍 中国香港

| 常见运营商 | 境内可直连骨干网 | 连接质量说明 |
| :--- | :--- | :--- |
| **CMI** (AS58453) | 全网（高性价比） | 移动自己的国际骨干网，境内连接质量非常高，只要对方接入足够 Transit，三网访问优先级高。 |
| **CUG** (AS10099) | 全网（联通优先） | 联通自己的国际骨干网，同 CMI，联通用户访问优先级最高。 |
| **NTT** | AS4809 (CN2)、AS58453 (CMI) | CN2 到 NTT 有时不可靠，延时抖动剧烈。移动高峰期直连可能被 QoS，速度不佳。 |
| **PCCW/HKT** | AS4134、AS4809、AS4837、AS9929、AS58453 | 综合连接效果一般。联通 A 网 (AS9929) 连接质量最好。移动 CMI 路由不稳定，不推荐。 |
| **Telstra Global** | AS4134、AS4837、AS4538、AS9929、AS58453 | **三网都可直连**，非常友好。电信 163 在香港地区直连 Telstra 效果顶级，推荐。移动 CMI 互联带宽较小，高峰延时可能猛涨。 |
| **CHT** | AS4134、AS4837、AS58453 | **电信 163** 效果极差，不推荐。**联通 169** 表现较好。移动 CMI 高峰表现极度拉垮。 |
| **HKBN/WTT** | AS4134、AS4837、AS58453 | 三网直连，主要用于流媒体解锁。无特殊优势。 |
| **HE** | AS58453 (经 HKIX)、AS9929 (via HKIX) | 电信、联通 169 和教育网都会**绕美**。 |
| **TaTa** | AS4837、AS9929 | 联通直连，大部分情况下回程绕美。**爆炸绕路三幻神之首**。 |
| **Cogentco** | 无 | **爆炸绕路三幻神之一**。电信、联通 9929、移动和教育网**绕美**，联通 169 **绕新加坡**。 |
| **GTT** | 无 | 三网均会**绕美**。 |

> **香港 VPS 总结：**
> * 黄金搭档：移动直连 + 电信直连。
> * 青睐香港的原因：奈飞港区字幕/配音优化、软件 APP 简体中文优化、翻墙节点集中。

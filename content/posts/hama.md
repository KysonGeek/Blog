## 📝 笔记方法：康奈尔笔记法 (Cornell Notes System)

康奈尔笔记法是由美国康奈尔大学教授沃尔特·波克发明的一种笔记方法。

- **目的:** 将笔记纸分为3个栏目：笔记、关键词、总结，来高效记笔记，并便于复习巩固知识点。
- **操作步骤:**
    1. &nbsp;&nbsp;用简洁的语句在**笔记栏**记下重点知识。
    2. &nbsp;&nbsp;将笔记栏中的知识点提炼为**关键词**和要点，写在关键词栏。在**总结栏**写下本次学习总结。
    3. &nbsp;&nbsp;遮盖笔记栏的内容，通过关键词栏的要点概括，复述出笔记内容来巩固知识。

## 📚 导读

这是一本入门级心理学书籍，旨在帮助患有抑郁的人走出困境。

蛤蟆本是一个热情、时尚又爱冒险的家伙，惹出过不少麻烦和笑话。但他现在陷入抑郁，不能自拔。他的朋友们（獾、河鼠、鼹鼠）决定督促他重视这个问题，并带他去接受心理咨询。

## 📌 笔记内容提炼（关键词与要点）

| 关键词                        | 笔记栏要点                                                   |
| :---------------------------- | :----------------------------------------------------------- |
| **心理咨询 / 自发做事**       | 1. 心理咨询是一个**自发的过程**，咨询师和来访者双方都得出于自愿。只有为自己而非取悦朋友才想咨询的时候，双方才能真正合作。<br>2. 咨询师相信每个人都有能力变得更好。 |
| **儿童自我状态 / 行为**       | 3. 处于**“儿童自我状态”**时，行为和感受都**像一个孩子**，不同于“幼稚”。<br>4. **自然型儿童**的基本情感：快乐和深情、愤怒、悲伤、恐惧。<br>5. 感受和行为与**小时候的自己如出一辙**，与实际年龄无关。<br>6. 早期的经历塑造了人格的雏形，无法否认或忘记。<br>7. **适应型儿童**情感：顺从行为（同意、取悦父母、道歉、依赖父母）。<br>8. 成长的本质是减少并最终打破对父母的依赖，成为一个独立自主的人。<br>9. **撒泼**是表达愤怒的**幼稚的短暂**方式。<br>10. **怄气**是表达愤怒的**幼稚的长期**方式。 |
| **儿童状态的后果**            | 11. 成人处在儿童状态时，道德层面上**没有“错”**，但会导致：被人嘲笑/尴尬，并让人觉得是**失败者**。<br>12. **“共谋”**：无意识地配合对方制造不快，是在玩心理游戏，输的人才是赢家。 |
| **学习是痛苦的**              | 13. 转变是痛苦的，因为看到自己不一定总是喜欢自己的样子。<br>14. 从当下变成想成为的自己，需要经历行为和态度的转变，付出艰辛的努力、勇气和决心。 |
| **父母自我状态 / 挑剔型父母** | 15. **“父母自我状态”**：包含从父母那里学到的所有**价值观和道德观**，以及对生活的评判标准。<br>16. **挑剔型父母**：爱批评人、愤怒、严厉。<br>17. **没有一种批判比自我批判更强烈**，也没有一个法官比我们自己更严苛。 |
| **成人状态 / 学习知识**       | 18. **“成人状态”**：指用**理性而不是情绪化**的方式来行事，应对此时此地的现实状况，能计划、考虑、决定、行动。<br>19. 在此状态下，知识和技能能为自己所用，**不被过去的声音或童年情绪驱使**。<br>20. 三种状态都是必需的，但**“成人状态”有它的特殊重要性**。<br>21. **只有在“成人自我状态”里，才能学到关于自我的新知识**。<br>22. 在“儿童状态”时，会再现过去的情绪，但学不到任何新的东西。<br>23. 在“父母状态”时，重复旧的观念和价值观，没有新知识和新理念的空间。<br>24. 进入成人状态难，是因为需要**艰辛的努力和刻意的思考**。 |
| **自己选择自己的感受**        | 25. **没有人能“让”我们产生什么感受**，是我们**“选择”了自己的感受**（愤怒或悲伤）。<br>26. **责怪**是人处在**“儿童自我状态”**里做的事情。<br>27. **“成人自我状态”**的人会**负起责任**。 |
| **自证预言**                  | 28. **“自证预言”**：我们会控制事件的发生，从而确保预言会成真，确保自己的世界和预期的一样。 |
| **人生坐标 / 必然后果**       | 29. 人生最早期的阶段（出生到大约四五岁）发生的一切对我们影响重大。<br>30. 核心问题：**“我是怎么看自己的？我好吗？”** 和 **“我是怎么看别人的？他们好吗？”**。<br>31. **“我不好，你好”**：**“受害者游戏”**、“PLOM: poor little old me"。 <br>32. **“我好，你不好”**：处于**父母自我状态**，常玩“Now I've Got You, You Son of Bitch”、“你为什么总让我失望？”。<br>33. **“我好；你也好”**：选择了这个坐标是许下了**终生的承诺**，人生坐标是动态的过程。 |
| **情商**                      | 34. **情商**：理解你内心的情感世界，并且还能掌控它。<br>35. **高情商**：有强大的**自我意识**；能**管理情绪**；能**控制冲动**，懂得**延迟满足**。 |

## ✨ 总结栏（总结、反思与观点）

1.  每个人都有儿童自我状态、父母自我状态。
2.  如果不对其他人经常生气，可能就是经常**对自己生气**（否定自己）。
3.  **在成人状态下才会学习到新的知识**，因为在其他两种状态下，我们有例可循（基础情感、父母影响），不需要思考。
4.  **幼时父母的“暴力”**，我们要选择原谅，原因有很多（父母是否是故意施暴，父母是否健在…………..）。
5.  处在父母自我状态下的人**不会抑郁**。他们觉得自己无敌好，别人无敌差，这样的人没心没肺。
6.  对一个人从依赖转变成了**对抗**，说明我们**成长了**。
7.  现在大环境下，大家普遍关注、研究物理世界，对于**情商世界知之甚少**。“不在沉默中爆发，就在沉默中死亡”这也适用于情感世界。
8.  怎样成为高情商的人：**客观看问题，三思而后行**。
